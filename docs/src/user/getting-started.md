# Getting Started

> **Note:** BookBoss is in early development. This guide will expand as features are completed.

## First Run

On first launch, BookBoss will detect that no admin account exists and prompt you to create one.
You will need to provide:

- A username
- An email address
- A password

This account becomes the administrator of your BookBoss instance.
